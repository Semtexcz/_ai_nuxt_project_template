# Typography

Typography should create hierarchy without adding visual noise.

## Fonts

- Heading font: `Nerko One`, used for headings only.
- Body font: system UI stack or Inter.
- Use at most two font families in the product.

## Hierarchy

- Use one `h1` per screen.
- Keep headings short and tied to the user task.
- Use body text for explanations, helper copy and validation messages.

## Readability

- Keep readable line length around 60-80 characters for long text.
- Avoid viewport-based font sizing.
- Keep line height comfortable for body copy and compact for controls.
